05BC3503 - WEB APPLICATION DEVELOPMENT USING PHP

This repository contains PHP programs created for academic learning and practical implementation of PHP concepts. The programs cover basic to intermediate PHP topics and are designed for educational purposes.

## How to Run

1. Install Laragon or XAMPP.
2. Place the project folder inside the `www` or `htdocs` directory.
3. Start Apache server.
4. Open your browser and visit:

```
http://localhost/WADU-PHP/
```

5. Run the required PHP file.
